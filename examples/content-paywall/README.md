# Content Paywall - Pay-Per-View with Stakefy x402

OnlyFans/Patreon style pay-per-view content.

## Features
- Pay once, view forever
- Preview before purchase
- Social payments (@username tips)
- Content unlock tracking

## Usage
```typescript
import { payToX } from 'x402-stakefy-sdk';

// Tip creator by username
await payToX(client, '@creator', 0.5);
```
